# AuthController

Контроллер для получения информации об авторизованном пользователе.

#### Методы

##### invoke

```php
public function __invoke(): JsonResponse
```

Возвращает текущего авторизованного пользователя вместе с его ролями и правами.

Вход: –

Выход: JSON-ответ:

```json
{
  "user": { ... },
  "permissions": [ ... ],
  "role": { ... }
}
```
